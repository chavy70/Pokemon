Motor de Base de Datos: SQL Server

1- Cambiar la cadena de conexión de SQL Server del archivo "appsettings.json", de la sección  "ConnectionStrings -> "DefaultConnection" por la de su Server local
por ejemplo:

"ConnectionStrings": {
  "DefaultConnection": "Server=localhost;Database=Pokemons;Trusted_Connection=True;"
}

O si usas autenticación de SQL Server:
"DefaultConnection": "Server=localhost;Database=Pokemons;User Id=sa;Password=tuPassword;"

2- Correr el Script de la carpeta Scripts/Crear_base_datos.sql para la creación de la base de datos a ser utilizada

3- Restaurar paquetes NuGet:
En Visual Studio, ir a Herramientas → Administrador de paquetes NuGet → Consola del administrador de paquetes.

Ejecutar:
Update-Package -reinstall

4- Aplicar las migraciones y crear la base de datos:

En la consola de NuGet, ejecutar:
Update-Database

5- Ejecutar el proyecto
